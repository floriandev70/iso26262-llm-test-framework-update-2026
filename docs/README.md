# docs

This directory stores study documentation and manuscript-related artifacts for the controlled replication repository.

## Current contents

- `LLAIS1_paper_26.pdf` — current manuscript artifact associated with the study.
- `REPLICATION_SPEC.md` — formal definition of what must be replicated from the historical baseline before extension experiments are considered valid.
- `PAPER_TO_REPO_MAPPING.md` — mapping from paper-level concepts to repository artifacts, planned artifacts, and evidence structures.
- `THREATS_TO_VALIDITY.md` — initial validity-risk register for the controlled replication and extension study.

## Documentation policy

- Markdown documents in this repository should stay aligned with the manuscript's framing when that framing is supported by repository evidence.
- If the manuscript and repository evidence diverge, repository evidence takes precedence until the discrepancy is resolved explicitly.
- Publication-oriented wording must remain traceable to concrete experiment configurations, prompts, logs, and preserved outputs.
